# Data Dictionary

The analysis expects a firm-quarter panel with one row per firm-quarter.

## Core Columns

| Column | Description |
| --- | --- |
| `firm_id` | Firm identifier used for synthetic data and diagnostics |
| `permno` | CRSP PERMNO or synthetic equivalent |
| `qdate` | Quarter-end date |
| `year` | Calendar year |
| `russell_index` | `R1000` or `R2000` |
| `rank_mktcap` | Within-index market-cap rank near the cutoff |
| `r1000_dummy` | Instrument equal to 1 for Russell 1000 observations |
| `cutoff_running_rank` | Constructed running variable; generated if absent from raw data |
| `distance_to_cutoff` | `cutoff_running_rank - 1000`; generated by the analysis script |
| `distance_to_cutoff_sq` | Squared distance to the cutoff; generated by the analysis script |
| `distance_x_r1000` | Side-specific local linear term; generated by the analysis script |
| `distance_sq_x_r1000` | Side-specific quadratic term; generated for sensitivity checks |
| `passive_pct_float` | Passive mutual fund ownership as share of float |
| `xrdq_change` | R&D change measure used as a continuous outcome |
| `d_xrdq_scaled` | Change in scaled R&D intensity, when available |
| `rd_cut` | Derived indicator equal to 1 when `xrdq_change < 0` |
| `rd_scaled_cut` | Derived indicator equal to 1 when `d_xrdq_scaled < 0` |

## Controls

The baseline controls are:

`atq`, `saleq`, `seqq`, `cheq`, `leverage`, `cf`, `rd_intensity`,
`sales_growth`, `log_analyst_coverage`, `BM`, `net_income_q`, `ROA`, `ROE`,
`cash_ratio`, and `stock_return_volatility`.

The expanded `full` control set also uses any columns beginning with `sic2_` as
industry fixed-effect dummies and any columns beginning with `year_` as year
fixed-effect dummies.
